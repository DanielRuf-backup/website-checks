# website-checks

`website-checks` checks websites with multiple services.

These are currently:
* crt.sh
* CryptCheck
* HSTS Preload List
* HTTP Observatory
* Lightouse
* PageSpeed Insights
* Security Headers
* sonarwhal
* SSL Decoder
* SSLLabs
* webbkoll

## Installation

`npm i -g website-checks`

## Usage

`website-checks example.com`
